# Hospital-Emergency-Room-Analytics-Dashboard

## Project Overview

This project analyzes Emergency Room (ER) operations using Power BI to provide insights into patient flow, admissions, wait times, referrals, demographics, and overall healthcare performance.

The dashboard helps stakeholders monitor operational efficiency, identify trends, and support data-driven decision-making.

## Tools & Technologies

* Power BI
* DAX
* Excel
* Data Visualization

## Dashboard KPIs

* Total Patients
* Average Wait Time
* Patient Satisfaction Score
* Admission Rate
* Referral Rate

## Key Insights

* Patient flow and admission status analysis
* Department-wise referral patterns
* Age group and gender distribution
* Wait time and satisfaction monitoring
* Operational performance tracking

## Skills Demonstrated

* Healthcare Analytics
* Dashboard Development
* Data Visualization
* DAX Calculations
* KPI Reporting
* Data Storytelling
